# SKILL: Guardián de Marca (Westside)

## 🎯 Objetivo de este Skill
Este conocimiento instruye al agente de Antigravity sobre las reglas visuales, de código y de diseño estricto para el desarrollo web de la marca "Westside" (Ropa de estilo propio y Barbería). 

## 📏 Reglas Generales
* **NUNCA** utilices frameworks CSS como Tailwind, Bootstrap o Bulma a menos que el usuario lo solicite explícitamente. Todo el diseño debe hacerse con CSS3 puro.
* **Lenguaje:** La página y sus componentes deben estar siempre en Español (Argentina).
* **Motor de Plantillas:** Utiliza la sintaxis oficial de Laravel Blade (`@if`, `@foreach`, `{{ $variable }}`) para todas las vistas.

## 🎨 Identidad Visual (Brand Guidelines)
Cuando generes código HTML/CSS, respeta esta estética:
* **Vibra:** Urbana, moderna, limpia, de estilo callejero (streetwear) combinado con la estética clásica de una barbería premium.
* **Paleta de Colores (Base):** * Fondo principal: Oscuro (`#121212` o similar) o Blanco roto, dependiendo de la sección.
  * Acentos: Usa colores que resalten pero mantengan sobriedad (ej. tonos grises, blanco puro para contrastar).
* **Tipografías:** Usa fuentes modernas y sin serifas (Sans-serif) tipo *Inter*, *Roboto* o tipografías gruesas para los títulos (tipo *Impact* o *Montserrat*).

## 🧩 Estructura de Componentes
* La sección de **Catálogo de Ropa** debe priorizar imágenes grandes y grillas responsivas (CSS Grid/Flexbox).
* La sección de **Barbería** debe ser clara, mostrando servicios, precios y (si aplica) un botón destacado para reservas.

## 🤖 Instrucción para el Agente
Antes de proponer o escribir cualquier código visual (HTML, Blade, CSS), revisa internamente este documento para asegurar que el diseño cumple con la identidad de Westside.
